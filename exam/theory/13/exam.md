# Динамическое программирование. Задача о рюкзаке (варианты со стоимостями и без).

## Задача о рюкзаке

Дано N предметов, ni предмет имеет массу wi > 0 и стоимость pi > 0. Необходимо выбрать такой набор предметов, чтобы
суммарная масса не превосходила заданной величины W, а суммарная стоимость была максимальна.

## Формулировка задачи

Дано N предметов, W -- вместимость рюкзака, w = {w1, ..., wN} -- веса, p = {p1, .., pN} -- стоимости предметов. Нужно
найти набор бинарных величин B = {b1, ..., bN}, bi = 1, если предмет ni включен в набор, bi = 0, если предмет ni не
включен, и такой что

- b1*w1 + ... + bNwN <= W
- b1*p1 + ... + bNpN максимальная

## Метод динамического программирования

Пусть A(k, s) есть максимальная стоимость предметов, которые можно уложить в рюкзак вместимости s, если можно
использовать только первые k предметов {n1, ..., nk} -- допустимые предметы для A(k, s)

A(k, 0) = 0
A(0, s) = 0

Найдем A(k, s). Есть два варианта

- Если предмет k не попал в рюкзак. Тогда A(k, s) равно максимальной стоимости рюкзака с такой же вместимостью и набором
  допустимых предметов {n1, ..., nk}, то есть A(k, s) = A(k-1, s)
- Если предмет k попал в рюкзак. Тогда A(k, s) = max(A(k-1, s), A(k-1, s-wk) + pk)

Асимптотика O(N*W)
